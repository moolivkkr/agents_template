# /worklog — Consolidated Project Ledger

**The problem this solves.** A pipeline run produces dozens of correct-but-siloed artifacts
(manifests, gate scores, ~25 reports/phase, decision-log, debate verdicts, ADRs, lessons). No single
place answers the three questions a human or a *new session* actually asks:

1. **What was implemented?** (per phase, per wave)
2. **What was decided, and why?** (ADRs, debate verdicts, spec deviations, gray-area picks)
3. **What is still pending / deferred / known-broken?**

`/worklog` **consolidates existing artifacts** (it does not capture new data) into
`docs/WORKLOG.md` — checked into git, regenerated idempotently, never hand-edited. It is
read-only over the pipeline state.

---

## What it reads (all already produced by the pipeline)

| Section | Source artifacts |
|---|---|
| Implemented | `agent_state/phases/*/manifest.json` → `artifacts.*`, `brd_requirements_met`; `PHASE_SUMMARY.md` |
| Decisions | `docs/DECISIONS.md` (spine), `docs/adr/`, `agent_state/debates/*-verdict.json`, `agent_state/phases/*/decision-log.md` |
| Pending / Deferred | manifest `known_issues[]` + `carried_forward[]`, `test_case_inventory.deferred_ids[]`, `agent_state/debates/unresolved.json`, FORCED gate markers, `agent_state/accept/unresolved_gaps.md` |
| Activity | `agent_state/phases/*/execution.jsonl`, `checkpoints/`, `roster.json` (skipped/not_applicable agents) |

## Procedure

The PARENT session runs this directly (no subagent needed — it's pure consolidation).

```bash
OUT="docs/WORKLOG.md"
PHASES=$(ls -d agent_state/phases/*/ 2>/dev/null | grep -oE '[0-9]+' | sort -n)
[ -n "${1:-}" ] && PHASES="$1"   # optional single-phase arg

python3 - "$OUT" $PHASES << 'PY'
import json, os, sys, glob, datetime
out = sys.argv[1]
phases = sys.argv[2:]
L = []
L.append("# Project Worklog — consolidated ledger")
L.append("")
L.append("> Generated by `/worklog`. **Do not hand-edit** — regenerate after each phase. Source of")
L.append("> truth for *what was built, decided, and deferred* across all agents and phases.")
L.append("")

def load(p):
    try: return json.load(open(p))
    except Exception: return None

for ph in phases:
    d = f"agent_state/phases/{ph}"
    m = load(f"{d}/manifest.json") or {}
    gate = "PASSED" if os.path.exists(f"{d}/gate.passed") else ("FORCED" if os.path.exists(f"{d}/gate.forced") else "in-progress")
    goal = m.get("goal") or m.get("phase_goal") or "—"
    L.append(f"## Phase {ph} — {goal}   [gate: {gate}]")
    L.append("")

    # 1) Implemented
    L.append("### Implemented")
    reqs = m.get("brd_requirements_met") or []
    if reqs: L.append(f"- Requirements met: {', '.join(reqs)}")
    arts = m.get("artifacts") or {}
    for k in ("code","api_routes","migrations","schemas","components"):
        v = arts.get(k)
        if v: L.append(f"- {k}: {', '.join(v) if isinstance(v,list) else v}")
    ps = f"{d}/PHASE_SUMMARY.md"
    if os.path.exists(ps): L.append(f"- Wave narrative: [{ps}]({ps})")
    if len(L) and L[-1].startswith("### Implemented"): L.append("- (no manifest artifacts recorded)")
    L.append("")

    # 2) Decisions — pointers merged from all sinks
    L.append("### Decisions")
    for v in sorted(glob.glob(f"{d.rsplit('/',2)[0] if False else 'agent_state/debates'}/*-verdict.json")):
        vj = load(v) or {}
        L.append(f"- [DEBATE] {vj.get('verdict_label', os.path.basename(v))} — {vj.get('decisive_factor','')} ({v})")
    for adr in sorted(glob.glob("docs/adr/ADR-*.md")):
        L.append(f"- [ADR] {os.path.basename(adr)} ({adr})")
    dl = f"{d}/decision-log.md"
    if os.path.exists(dl): L.append(f"- [DEVIATIONS] see {dl}")
    if L[-1].startswith("### Decisions"): L.append("- (none recorded)")
    L.append("")

    # 3) Pending / Deferred — merged from all sinks
    L.append("### Pending / Deferred")
    for item in (m.get("carried_forward") or []):
        L.append(f"- carried_forward: {item}")
    for item in (m.get("known_issues") or []):
        L.append(f"- known_issue: {item}")
    tci = m.get("test_case_inventory") or {}
    if tci.get("deferred_ids"): L.append(f"- deferred TC: {', '.join(tci['deferred_ids'])}")
    if gate == "FORCED": L.append("- ⚠ gate was FORCED — see gate.forced for the override reason")
    if L[-1].startswith("### Pending"): L.append("- (none)")
    L.append("")

    # 4) Activity
    L.append("### Activity")
    ex = f"{d}/execution.jsonl"
    if os.path.exists(ex):
        agents, failed = set(), 0
        for line in open(ex):
            try:
                e = json.loads(line)
                if e.get("status") == "completed": agents.add(e.get("agent"))
                if e.get("status") == "failed": failed += 1
            except Exception: pass
        L.append(f"- {len(agents)} agents completed, {failed} failures")
    roster = load(f"{d}/roster.json") or {}
    skipped = [a for a,x in (roster.get('agents') or {}).items() if x.get('status') != 'required']
    if skipped: L.append(f"- explicitly skipped/N-A: {', '.join(skipped)}")
    L.append("")

# Global backlog roll-up
gaps = "agent_state/accept/unresolved_gaps.md"
unresolved = "agent_state/debates/unresolved.json"
L.append("## Global Backlog")
if os.path.exists(gaps): L.append(f"- Acceptance gaps: [{gaps}]({gaps})")
uj = load(unresolved)
if uj: L.append(f"- Auto-resolved decisions needing review: {unresolved}")
if L[-1] == "## Global Backlog": L.append("- (clean)")
L.append("")

open(out, "w").write("\n".join(L) + "\n")
print(f"✓ Wrote {out} covering phases: {', '.join(phases) or '(none)'}")
PY
```

## When to run
- After each phase gate (the orchestrator Post-Gate step calls it automatically — see
  `develop-orchestrator.md`).
- Before a stakeholder review or handoff.
- At the start of a NEW session to get oriented (or just read the committed `docs/WORKLOG.md`).

## Relationship to other surfaces
- `/status` = live gate/test/git snapshot (ephemeral, recomputed each run).
- `docs/DECISIONS.md` = the durable decision spine (Tier 0.5) — `/worklog` renders it in context.
- `docs/WORKLOG.md` = the consolidated human-readable ledger (this command's output).
