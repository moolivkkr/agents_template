# CRUD Service Test Archetype (Rust / mockall)

Complete unit test template for the Rust service layer. Every generated service test MUST follow this pattern.

## Test Module Location

```
src/
  services/
    widget.rs            <- production code
    widget.rs            <- #[cfg(test)] mod tests at bottom of same file
  traits/
    repository.rs        <- WidgetRepository trait with #[automock]
    cache.rs             <- Cache trait with #[automock]
    audit.rs             <- AuditWriter trait with #[automock]
```

Rule: Unit tests live inside `#[cfg(test)] mod tests` at the bottom of the production file. Mocks are auto-generated by `mockall` from trait definitions.

## Trait Definitions with #[automock]

```rust
// src/traits/repository.rs
use async_trait::async_trait;
use mockall::automock;
use uuid::Uuid;

use crate::domain::{ListFilters, ListResult};
use crate::error::AppError;
use crate::models::Widget;

#[automock]
#[async_trait]
pub trait WidgetRepository: Send + Sync {
    async fn create(&self, widget: &Widget) -> Result<(), AppError>;
    async fn get_by_id(&self, tenant_id: Uuid, id: Uuid) -> Result<Widget, AppError>;
    async fn update(&self, widget: &Widget) -> Result<(), AppError>;
    async fn soft_delete(&self, tenant_id: Uuid, id: Uuid) -> Result<(), AppError>;
    async fn list(
        &self,
        tenant_id: Uuid,
        filters: &ListFilters,
    ) -> Result<ListResult<Widget>, AppError>;
}

// src/traits/cache.rs
use async_trait::async_trait;
use mockall::automock;
use std::time::Duration;

use crate::error::AppError;

#[automock]
#[async_trait]
pub trait Cache: Send + Sync {
    async fn get(&self, key: &str) -> Result<Option<Vec<u8>>, AppError>;
    async fn set(&self, key: &str, value: &[u8], ttl: Duration) -> Result<(), AppError>;
    async fn delete(&self, key: &str) -> Result<(), AppError>;
}

// src/traits/audit.rs
use async_trait::async_trait;
use mockall::automock;

use crate::domain::AuditEntry;
use crate::error::AppError;

#[automock]
#[async_trait]
pub trait AuditWriter: Send + Sync {
    async fn write(&self, entry: AuditEntry) -> Result<(), AppError>;
}
```

## Test Factory Pattern

```rust
#[cfg(test)]
mod tests {
    use super::*;
    use chrono::Utc;
    use mockall::predicate::*;
    use std::sync::Arc;
    use uuid::Uuid;

    use crate::error::AppError;
    use crate::models::Widget;
    use crate::traits::cache::MockCache;
    use crate::traits::repository::MockWidgetRepository;
    use crate::traits::audit::MockAuditWriter;

    // --- Test Factories ---

    /// Build a Widget with sensible defaults. Override fields with builder methods.
    fn make_widget() -> Widget {
        let now = Utc::now();
        Widget {
            id: Uuid::new_v4(),
            tenant_id: Uuid::new_v4(),
            name: format!("widget-{}", &Uuid::new_v4().to_string()[..8]),
            description: Some("A test widget".into()),
            status: "active".into(),
            created_at: now,
            updated_at: now,
            deleted_at: None,
            created_by: Uuid::new_v4(),
            updated_by: Uuid::new_v4(),
            version: 1,
        }
    }

    fn make_widget_with_tenant(tenant_id: Uuid) -> Widget {
        let mut w = make_widget();
        w.tenant_id = tenant_id;
        w
    }

    fn make_widget_with_version(tenant_id: Uuid, version: i32) -> Widget {
        let mut w = make_widget_with_tenant(tenant_id);
        w.version = version;
        w
    }

    fn make_create_input() -> CreateWidgetInput {
        CreateWidgetInput {
            name: "New Widget".into(),
            description: Some("A fine widget".into()),
        }
    }

    fn make_update_input(version: i32) -> UpdateWidgetInput {
        UpdateWidgetInput {
            name: "Updated Widget".into(),
            description: Some("Updated description".into()),
            version,
        }
    }

    /// Build the service under test with fresh mock dependencies.
    fn setup() -> (
        WidgetService,
        Arc<MockWidgetRepository>,
        Arc<MockCache>,
        Arc<MockAuditWriter>,
        Uuid, // tenant_id
        Uuid, // user_id
    ) {
        let repo = Arc::new(MockWidgetRepository::new());
        let cache = Arc::new(MockCache::new());
        let audit = Arc::new(MockAuditWriter::new());
        let tenant_id = Uuid::new_v4();
        let user_id = Uuid::new_v4();

        let svc = WidgetService::new(
            repo.clone() as Arc<dyn WidgetRepository>,
            cache.clone() as Arc<dyn Cache>,
            audit.clone() as Arc<dyn AuditWriter>,
        );

        (svc, repo, cache, audit, tenant_id, user_id)
    }

    // NOTE: Because mockall generates mock structs behind Arc, we need
    // Arc::get_mut to set expectations before the service consumes the Arc.
    // Alternative: use setup_with_expectations pattern shown below.

    /// Build service with pre-configured mock expectations.
    fn setup_service(
        configure_repo: impl FnOnce(&mut MockWidgetRepository),
        configure_cache: impl FnOnce(&mut MockCache),
        configure_audit: impl FnOnce(&mut MockAuditWriter),
    ) -> (WidgetService, Uuid, Uuid) {
        let mut repo = MockWidgetRepository::new();
        let mut cache = MockCache::new();
        let mut audit = MockAuditWriter::new();
        let tenant_id = Uuid::new_v4();
        let user_id = Uuid::new_v4();

        configure_repo(&mut repo);
        configure_cache(&mut cache);
        configure_audit(&mut audit);

        let svc = WidgetService::new(
            Arc::new(repo),
            Arc::new(cache),
            Arc::new(audit),
        );

        (svc, tenant_id, user_id)
    }
```

## Get Tests (Cache Hit / Miss)

```rust
    // --- Get Tests ---

    #[tokio::test]
    async fn get_cache_hit() {
        let tenant_id = Uuid::new_v4();
        let widget = make_widget_with_tenant(tenant_id);
        let widget_id = widget.id;
        let cached_data = serde_json::to_vec(&widget).unwrap();

        let (svc, _, _) = setup_service(
            |repo| {
                // DB must NOT be called on cache hit
                repo.expect_get_by_id().times(0);
            },
            |cache| {
                let data = cached_data.clone();
                cache.expect_get()
                    .withf(move |key: &str| key.contains(&tenant_id.to_string()))
                    .times(1)
                    .returning(move |_| Ok(Some(data.clone())));
            },
            |_audit| {},
        );

        let result = svc.get(tenant_id, widget_id).await;
        let fetched = result.expect("get should succeed from cache");
        assert_eq!(fetched.id, widget_id);
    }

    #[tokio::test]
    async fn get_cache_miss_queries_db_and_populates_cache() {
        let tenant_id = Uuid::new_v4();
        let widget = make_widget_with_tenant(tenant_id);
        let widget_id = widget.id;
        let widget_clone = widget.clone();

        let (svc, _, _) = setup_service(
            |repo| {
                let w = widget_clone.clone();
                repo.expect_get_by_id()
                    .withf(move |tid: &Uuid, id: &Uuid| *tid == tenant_id && *id == widget_id)
                    .times(1)
                    .returning(move |_, _| Ok(w.clone()));
            },
            |cache| {
                // Cache miss
                cache.expect_get()
                    .times(1)
                    .returning(|_| Ok(None));
                // Cache set after DB fetch
                cache.expect_set()
                    .times(1)
                    .returning(|_, _, _| Ok(()));
            },
            |_audit| {},
        );

        let result = svc.get(tenant_id, widget_id).await;
        let fetched = result.expect("get should succeed from DB");
        assert_eq!(fetched.id, widget_id);
    }

    #[tokio::test]
    async fn get_not_found() {
        let tenant_id = Uuid::new_v4();
        let widget_id = Uuid::new_v4();

        let (svc, _, _) = setup_service(
            |repo| {
                repo.expect_get_by_id()
                    .times(1)
                    .returning(move |_, id| Err(AppError::NotFound {
                        resource: "widget".into(),
                        identifier: id.to_string(),
                    }));
            },
            |cache| {
                cache.expect_get()
                    .times(1)
                    .returning(|_| Ok(None));
            },
            |_audit| {},
        );

        let result = svc.get(tenant_id, widget_id).await;
        assert!(result.is_err());
        assert!(matches!(result.unwrap_err(), AppError::NotFound { .. }));
    }

    #[tokio::test]
    async fn get_cache_error_falls_through_to_db() {
        let tenant_id = Uuid::new_v4();
        let widget = make_widget_with_tenant(tenant_id);
        let widget_id = widget.id;
        let widget_clone = widget.clone();

        let (svc, _, _) = setup_service(
            |repo| {
                let w = widget_clone.clone();
                repo.expect_get_by_id()
                    .times(1)
                    .returning(move |_, _| Ok(w.clone()));
            },
            |cache| {
                // Cache returns error — should not fail the request
                cache.expect_get()
                    .times(1)
                    .returning(|_| Err(AppError::Internal("redis down".into())));
                // Still attempts to set (best effort)
                cache.expect_set()
                    .times(1)
                    .returning(|_, _, _| Ok(()));
            },
            |_audit| {},
        );

        let result = svc.get(tenant_id, widget_id).await;
        assert!(result.is_ok(), "cache error must not fail the request");
    }
```

```rust
                    .times(1)

            },

        let result = svc.update(tenant_id, user_id, existing_id, input.clone()).await;

        assert_eq!(widget.version, 2);
```

## Critical Rules

- Every test MUST use fresh mock instances — never share mock state between tests
- `mockall` `times(0)` MUST be used to assert methods are NOT called (e.g., repo not called on validation failure)
- `withf` MUST be used for specific argument assertions — avoid blanket `returning` without validation
- Cache hit test MUST verify DB is NOT queried (`expect_get_by_id().times(0)`)
- Cache miss test MUST verify DB IS queried AND cache IS populated
- Cache/audit errors MUST NOT fail the business operation — assert `result.is_ok()` after mock returns error
- Version conflict test: existing.version = 3, input.version = 1 — assert Conflict error
- Validation failures MUST prevent any repository calls (`expect_create().times(0)`)
- Audit log tests MUST verify action, entity_id, tenant_id, actor_id, and timestamp
- Wrong tenant MUST return NotFound, never Forbidden
- Use `#[tokio::test]` on every async test function
- Mock expectations are automatically verified when the mock is dropped — explicit `checkpoint()` is optional
