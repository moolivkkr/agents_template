## Grading Scale Reference

| Grade | Meaning | Usage |
|-------|---------|-------|
| **A** | Official product documentation (Help Center, Admin Guide, PDF manuals) | Authoritative, version-specific. Cite for all factual claims. |
| **B** | Official supplementary docs (best practices, integration guides, KB articles, API references) | Authoritative but narrower scope. Good for specific topics. |
| **C** | Marketing materials, data sheets, product briefs | Accurate but high-level. Use for overview, not configuration. |
| **D** | Community forums, blog posts, third-party reviews | Useful context but may be outdated or imprecise. Cross-reference. |
| **E** | Inferred from training materials or indirect references | Treat as tentative. Verify before citing. |
| **U** | Ungraded (videos, training courses) | Rated by relevance, not grade. Cross-reference with docs. |
